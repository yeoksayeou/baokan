Chinese Periodical Translation Project

A site for students and researchers to browse translations of Chinese newspapers. Translations were done with generative AI: they must be read with that in mind and handled with care.

人民日報
The People’s Daily

Issues from 1946-1972 and 1989.
申報
Shenbao

Issues from 1930-1937 and 1939-1949.

Learn more about this project here: About this Project. You can download a complete copy of this website and browse it much faster by downloading a copy of the repository on GitHub.

This website is an experiment. Mistakes and problems with the translations are not being manually fixed: being able to explore the imperfections of the automatically produced translations is one of the purposes of this website, but anyone is free to download, improve, and recreate the website with their own corrected versions of pieces. To explore some of the issues that have been identified with the translations here, explore them here: LLM Translation Issues
